## 🚀 Tentang Proyek
Aplikasi web mandiri super ringan yang dibuat murni menggunakan HTML, CSS, dan Vanilla JavaScript. Tanpa *framework*, tanpa *database* eksternal. Semua datamu (tugas, tautan, pengaturan) tersimpan aman secara lokal di komputermu sendiri menggunakan teknologi **Local Storage**.

---

## 🌟 Fitur Unggulan

<details>
  <summary><b>🕐 Sapaan & Waktu Dinamis (Klik untuk buka)</b></summary>
  <br>
  Menampilkan jam dan tanggal secara <i>real-time</i>. Sistem akan menyapamu sesuai dengan waktu saat ini (Pagi, Siang, Sore, Malam). Kamu juga bisa mengganti nama panggilanmu kapan saja!
</details>

<details>
  <summary><b>🍅 Timer Pomodoro Kustom (Klik untuk buka)</b></summary>
  <br>
  Tetap fokus dengan teknik Pomodoro! Timer bawaan diatur pada 25 menit, namun kamu bisa menyesuaikan durasinya mulai dari 1 hingga 120 menit. Dilengkapi tombol Mulai, Jeda, dan Reset.
</details>

<details>
  <summary><b>📝 Smart To-Do List (Klik untuk buka)</b></summary>
  <br>
  Catat semua tugas harianmu. Fitur ini dilengkapi pencegah duplikasi tugas dan otomatis mengurutkan tugas yang belum selesai di posisi paling atas.
</details>

<details>
  <summary><b>🔗 Quick Bookmarks (Klik untuk buka)</b></summary>
  <br>
  Simpan tautan situs web yang paling sering kamu kunjungi. Jika kamu lupa mengetik <code>https://</code>, sistem akan menambahkannya secara otomatis.
</details>

<details>
  <summary><b>🌗 Tema Glassmorphism (Klik untuk buka)</b></summary>
  <br>
  Mendukung <i>Light Mode</i> dan <i>Dark Mode</i> dengan efek kaca buram (Glassmorphism) dan latar belakang gradasi yang bergerak melayang secara estetik.
</details>

## ⚙️ Cara Menjalankan

Karena proyek ini **Zero-Dependency** (tidak butuh Node.js, NPM, atau server *backend*), cara menjalankannya sangat mudah!

1. Unduh atau *clone* proyek ini.
2. Pastikan ketiga file ini berada di dalam satu folder yang sama:
   - `index.html`
   - `style.css`
   - `script.js`
3. Klik ganda pada file `index.html` untuk membukanya di peramban (browser) pilihanmu (Chrome, Firefox, Safari, Edge).
4. 🎉 **Selesai!** Selamat menikmati produktivitasmu.
